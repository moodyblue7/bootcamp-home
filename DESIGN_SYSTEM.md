# 부트캠프 홈페이지 디자인 규칙

정부·지자체 평가자, 협력기관, 제조 도입기업, 투자자가 한 회사로 인식할 수 있도록
헬스케어와 제조 AX를 하나의 브랜드 체계로 묶는다.

## 1. 색상

| 역할 | 색상 | 사용 원칙 |
|---|---|---|
| 핵심 제목 | Navy `#17232E` | 모든 주요 제목의 기본색 |
| 본문 | Navy 600 `#3A4956` | 설명과 상세 문장 |
| 브랜드 강조 | Blue `#0B3FD8` | 회사명, 라벨, 링크, CTA 등 짧은 강조 |
| 헬스케어 보조 | Cyan `#00CFF0` | 데이터 선, 아이콘, 배경 장식에만 사용 |
| 배경 | White `#FFFFFF` | 기본 정보 화면 |

- 긴 제목 전체에 그라데이션을 사용하지 않는다.
- Cyan은 흰 배경 위 작은 글자색으로 사용하지 않는다.
- 헬스케어와 제조 AX의 차이는 글자색보다 사진, 도식, 데이터 그래픽으로 표현한다.
- 그라데이션은 데이터 흐름, 진행 표시, 작은 장식 요소로 제한한다.

## 2. 타이포그래피

| 역할 | 크기 | 행간 | 굵기 | 용도 |
|---|---:|---:|---:|---|
| Hero | 40–56px | 1.15 | 800 | 페이지 첫 핵심 메시지 |
| Section | 32–48px | 1.25 | 800 | 대주제 제목 |
| Card heading | 24px | 1.4 | 700–800 | 카드·세부 항목 제목 |
| Lead | 18–22px | 1.7 | 600 | 제목 직후 핵심 설명 |
| Body | 16–18px | 1.85 | 400 | 일반 설명 문단 |
| Label | 14px | 1.5 | 700 | ABOUT, TECHNOLOGY 등 영문 라벨 |

- 제목은 `S-Core Dream`, 본문은 `Freesentation`을 사용한다.
- 한 화면에는 Hero 또는 Section 제목을 하나만 둔다.
- 강조색은 한 제목 안에서 한 구절 이하로 제한한다.
- 본문은 한 줄이 지나치게 길어지지 않도록 최대 48rem로 제한한다.

## 3. 위치와 간격

- 콘텐츠 최대 폭: 72rem.
- 홈 플로우의 제목과 본문은 동일한 왼쪽 기준선에 맞춘다.
- 모바일 좌우 여백 24px, 태블릿 40px, 데스크톱은 왼쪽 탐색 영역을 포함해 176px로 맞춘다.
- 화면 위·아래 여백: 모바일 80px, 태블릿 96px, 데스크톱 112px.
- Label → 제목: 16px.
- 제목 → Lead: 32px.
- 문단 사이: 20px.
- 카피 → 대표 이미지: 32–40px.

## 4. 구현 클래스

- `.flow-panel`: 홈 플로우의 공통 폭과 여백
- `.flow-label`: 영문 섹션 라벨
- `.flow-hero-title`: 첫 화면 제목
- `.flow-section-title`: 일반 섹션 제목
- `.flow-lead`: 핵심 설명
- `.flow-body`: 일반 본문
- `.flow-link`: 텍스트 링크

먼저 홈 첫 화면과 회사소개 화면에 적용하고, 확인 후 나머지 화면으로 확장한다.

## 5. 연혁 화면

- 최근 연도가 위에 오도록 역순으로 배치한다.
- 최신 연도만 브랜드 블루와 옅은 배경으로 강조한다.
- 나머지 연도 숫자는 Navy를 사용하며, 긴 숫자·제목에 그라데이션을 사용하지 않는다.
- 연도별 내용을 하나의 경계 안에 묶고, 월 표기의 폭을 고정해 항목 제목의 시작선을 맞춘다.
- 핵심 성과는 굵기와 Blue 점으로만 구분하고 색상 종류를 늘리지 않는다.
